Aliceconv - The converter collections for old Alicesoft games

Purpose

The purpose of this toolkit is to analyze past games and convert them into a modern and easy-to-understand format.

Tools

  • Splitter
    • It extracts files from disk image or archive file.
    • N88Splitter
      • Extractor for N88 DISK BASIC formatted disk image.
      1. Create disk image with KryoFlux.
      2. Convert to D88 disk image using SAMDisk.
      3. Export IMG file (RAW Sector file format) using HxCFloppyEmulator software.
      4. command prompt> N88Spiltter <disk image>
    • DATSplitter
      • Splitter for Alicesoft archive file. Such as ACG.DAT, AMUS.DAT, ADISK.DAT etc. If the archive file has multi-archive link-information, Out-file will be named by that information. Wildcard allowed.
    • DATSplitter_old
      • Sometimes DATsplitter failed by very-old archive file. In that case, please use it. Wildcard allowed.
  • CG
    • VSPtoPNG
      • Main tool of CG converters. It automatically detects these formats.
      • GL - very early CG format (8/512 colors) from Champion soft era (Predecessor of Alicesoft) to almost all PC-8801 games (except RanceII).
      • GL3 - early CG format (16/4096 colors) form Rance to Alice no yakata CD. PC-9801, PC-88VA and almost FM TOWNS, X68000.
      • GM3 - variant of GL3 (8/512 colors). Intruder for FM TOWNS only.
      • VSP - (16/4096 colors). Supported by many titles for PC-9801/FM TOWNS/X68000.
      • VSP200line - variant of VSP (8/256 colors). Supported by RanceII for PC-8801 and Little Vampire for PC-9801 (ALICE-NET version).
      • VSP 256 colors - (256/16M colors). Format used in 256-colors games after Super D.P.S.
      • PMS - variant of VSP 256 colors. Format used after Kichikuou Rance (System 3.5).
      • RanceII for X68000
      • Tousin Toshi for X68000
      • Abunai Bunkasai Zenya for X68000 in RanceIII Option Set for X68000
      There are no common transparent information. So I cannot include transparent infos.
    • TIFFtoPNG_Y1CD
      • FM TOWNS TIFF (256 colors). Format used in Alice no Yakata CD, Alice no Yakata CD II, Rance III FM TOWNS.
    • MSXtoPNG
      • Converter for MSX2 CG.
    • MSXtoPNG_LP
      • for Little Princess (Champion Soft).
    • MSXtoPNG_LV
      • for Little Vampire (Champion Soft).
    • MSXtoPNG_DrSTOP
      • for Dr.STOP!.
    • MSXtoPNG_TousinToshi
      • for Tousin Toshi.
    • MSXtoPNG_Intruder
      • for Intruder. It needs COLPALET.DAT in A disk.
    • MSXtoPNG_GS, MSXtoPNG_GS_TITLE
      • Under development converter for Gakuen Senki (Champion Soft).
    • DRStoPNG
      • Converter for Dr.STOP! opening demo CG (cg003.dat) for PC-9801/X68000.
    • DRStoPNG_TOWNS
      • Converter for Dr.STOP! opening demo CG (cg003.dat) and openning.dat for FM TOWNS.
  • Sound
    • PC88toVGM
      • Converter for PC-8801SR MML format to VGM.
      • Little Vampire for PC-8801SR (Chanpion Soft)
      • Rance for PC-8801SR
      • Intruder for PC-8801SR
      • Abunai Tengu Densetu for PC-8801SR
      • Dream Program System for PC-8801SR
    • PSGtoVGM
      • Converter for MSX PSG DATA to VGM.
      • Rance for MSX2
      • Intruder for MSX2
      • Cresent Moon Girl for MSX2
      • Abunai Tengu Densetu for MSX2
      • Dream Program System for MSX2
      • RanceII for MSX2
      • Dream Program System -SG- for MSX2
      • Dream Program System -SG- set2 for MSX2
      • Dream Program System -SG- set3 for MSX2
    • EOMMLtoVGM
      • Converter for EOMML format to VGM.
      • Rance for X68000/PC-9801
      • Cresent Moon Girl for X68000/PC-9801
      • Abunai Tengu Densetu for X68000/PC-9801
      • Dream Program System for X68000/PC-9801
      • Alice no Yakata for X68000/PC-9801
      • RanceII for X68000
      • Dream Program System -SG- for X68000
      • Tousin Toshi for X68000
      • Dream Program System -SG- set2 for X68000
      • Dream Program System -SG- set3 for X68000
      It cannot autodetect machine specific information. I decided X68000 as default. It has 4 options.
      • -9 for PC-9801 (YM-2203) musics.
      • -8 for PC-8801 (YM-2203) musics. (I'll remove it)
      • -T for ToushinToshi for X68000 (YM-2151) musics.
      • -s(num) for SSG relative volume (1-255%) to FM.
    • MAKO1toVGM
      • Converter for MAKO1 format to VGM.
      • Rance for PC-88VA/FM TOWNS
      • Intruder for FM TOWNS (in Alice no Yakata CD)
      • Cresent Moon Girl for PC-88VA, FM TOWNS (in Alice no Yakata CD)
      • Abunai Tengu Densetu for PC-88VA/FM TOWNS
      • Dream Program System for PC-88VA/FM TOWNS
      • Alice no Yakata for PC-88VA
      • RanceII for PC-9801/PC-8801/FM TOWNS and may-be PC-88VA (I have not)
      • Dream Program System -SG- for PC-9801/PC-88VA/FM TOWNS
      • Tousin Toshi for PC-9801/FM TOWNS
      • Dream Program System -SG- set2 for PC-9801/PC-88VA/FM TOWNS
      • Alice no Yakata CD for FM TOWNS
      • Dream Program System -SG- set3 for PC-9801/PC-88VA/FM TOWNS
      It cannot autodetect machine specific information. I decided PC-9801 as default. so It has 3 options.
      • -v for PC-88VA (YM-2203) musics.
      • -t for FM TOWNS (YM-2612) musics.
      • -s(num) for SSG relative volume (1-255%) to FM.
      If NOT -v/-t options applied. It works for PC-9801 music.
    • MAKO2toVGM
      • Converter for MAKO2 format to VGM. It supports after RanceIII to Windows 3.1 era FM/SSG musics (except D.P.S -SG set3). It also converts SSG-PCM or TOWNS PCM to Windows WAVE. Autodetect file types (except RanceIV and Alice no Yakata 3). It has 5 options.
      • -n for OPN mode.
      • -x for OPM (X68000) mode.
      • -a for OPNA mode.
      • -d for first impliment (RanceIII). Correct Detune level.
      • -s(num) for SSG relative volume (1-255%) to FM.
