Aliceconv - The converter collections for old Alicesoft games

Purpose

The purpose of this toolkit is to analyze past games and convert them into a modern and easy-to-understand format.

Tools

  • Splitter
    • It extracts files from disk image or archive file.
    • N88Splitter
      • Extractor for N88 DISK BASIC formatted disk image.
      1. Create disk image with KryoFlux.
      2. Convert to D88 disk image using SAMDisk.
      3. Export IMG file (RAW Sector file format) using HxCFloppyEmulator software.
      4. command prompt> N88Spiltter <disk image>
    • DATSplitter
      • Splitter for Alicesoft archive file. Such as ACG.DAT, AMUS.DAT, ADISK.DAT etc. If the archive file has multi-archive link-information, Out-file will be named by that information. Wildcard allowed.
    • DATSplitter_old
      • Sometimes DATsplitter failed by very-old archive file. In that case, please use it. Wildcard allowed.
  • CG
    • VSPtoPNG
      • Main tool of CG converters. It automatically detects these formats.
      • GL - very early CG format (8/512 colors) from Champion soft era (Predecessor of Alicesoft) to almost all PC-8801 games (except RanceII).
      • GL3 - early CG format (16/4096 colors) form Rance to Alice no yakata CD. PC-9801, PC-88VA and almost FM TOWNS, X68000.
      • GM3 - variant of GL3 (8/512 colors). Intruder for FM TOWNS only.
      • VSP - (16/4096 colors). Supported by many titles for PC-9801/FM TOWNS/X68000.
      • VSP200line - variant of VSP (8/256 colors). Supported by RanceII for PC-8801 and Little Vampire for PC-9801 (ALICE-NET version).
      • VSP 256 colors - (256/16M colors). Format used in 256-colors games after Super D.P.S.
      • PMS - variant of VSP 256 colors. Format used after Kichikuou Rance (System 3.5).
      • FM TOWNS TIFF (256 colors). Format used in Alice no Yakata CD, Alice no Yakata CD II, Rance III FM TOWNS.
      • RanceII for X68000
      • Tousin Toshi for X68000
      There are no common transparent information. So I cannot include transparent infos.
