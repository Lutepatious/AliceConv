Aliceconv - The converter collections for old Alicesoft games

Warning

This toolset will be 64-bit version only. Ver. 1.51.1 is final 32-bit support version. (You can compile 32-bit version, I think.)

Purpose

The purpose of this toolkit is to analyze past games and convert them into a modern and easy-to-understand format.

Tools

  • Splitter
    • It extracts files from disk image or archive file.
    • N88Splitter
      • Extractor for N88 DISK BASIC formatted disk image.
      1. Create disk image with KryoFlux.
      2. Convert to D88 disk image using SAMDisk.
      3. Export IMG file (RAW Sector file format) using HxCFloppyEmulator software.
      4. command prompt> N88Spiltter <disk image>
    • DATSplitter
      • Splitter for Alicesoft archive file. Such as ACG.DAT, AMUS.DAT, ADISK.DAT etc. If the archive file has multi-archive link-information, Out-file will be named by that information. Wildcard allowed.
    • DATSplitter_old
      • Sometimes DATsplitter failed by very-old archive file. In that case, please use it. Wildcard allowed.
  • CG
    • toPNG
      • New and Unified CG converter. It has many options.
        X68000 version specific options.
      • -XP for Tousin Toshi (and hint disk) "SPRPT.SPR"
      • -XM for Tousin Toshi (and hint disk) "ATDAT.SPR"
      • -XR for RanceII (and hint disk)
      • -XT for Tousin Toshi 256 colors CG
      • -XB for Abunai Bunkasai zen'ya in RanceIII Option SET
        FM TOWNS version specific opitons.
      • -TS for Dr.STOP! "CG003.BMP"
      • -TO for Dr.STOP! "OPENNING.BMP"
      • -TY for TIFF variant since Alice no yakata CD
      • -TI for "*.ICN" TownsMENU icons
        MSX2 version specific options.
      • -Mb for MSX Screen7 BSAVE format (Gakuen Senki TITLE, Fukei san VX FUKEI.CG, Abunai Tengu Densetu README*.GE7, Dream Program System POPART?.DAT
      • -ML for Little Princess (Champion Soft).
      • -MV for for Little Vampire (Champion Soft).
      • -MG for Gakuen Senki (Champion Soft). needs RXX on current directory
      • -Mg for Rance, Cresent Moon Girl, Abunai Tengu Densetsu, Dream Pregram System, RanceII, Dream Program System -SG-.
      • -MI for Intruder. needs COLPALET.DAT on current directory
      • -MS for Dream Program System -SG- Interlace mode
      • -MT for Toushin Toshi.
      • -MD for Dr.STOP!.
        • Gakuen Senki needs RXX file on current directory. Please copy from any game disk. It contains palette and location information. Due to bugs and lacked palette, some images use previous image's palette.
      • -s for Dr.STOP! PC-9801 or X68000 version's "CG003.DAT"
      • -g for GL - very early CG format (8/512 colors) from Champion soft era (Predecessor of Alicesoft) to almost all PC-8801 games (except RanceII).
      • -h for GL3 - early CG format (16/4096 colors) form Rance to Alice no yakata CD. PC-9801, PC-88VA and almost FM TOWNS, X68000.
      • -r for VSP200line - variant of VSP (8/256 colors). Supported by RanceII for PC-8801 and Little Vampire for PC-9801 (ALICE-NET version).
      • -v for VSP - (16/4096 colors). Supported by many titles for PC-9801/FM TOWNS/X68000.

      • Converter for EOMML format to VGM.
      • Rance for X68000/PC-9801
      • Cresent Moon Girl for X68000/PC-9801
      • Abunai Tengu Densetu for X68000/PC-9801
      • Dream Program System for X68000/PC-9801
      • Alice no Yakata for X68000/PC-9801
      • RanceII for X68000

      • Converter for MAKO1 format to VGM.
      • Rance for PC-88VA/FM TOWNS
      • Intruder for FM TOWNS (in Alice no Yakata CD)
      • Cresent Moon Girl for PC-88VA, FM TOWNS (in Alice no Yakata CD)
      • Abunai Tengu Densetu for PC-88VA/FM TOWNS
      • Dream Program System for PC-88VA/FM TOWNS
      • Alice no Yakata for PC-88VA
      • RanceII for PC-9801/PC-8801/PC-88VA/FM TOWNS
      • Dream Program System -SG- for PC-9801/PC-88VA/FM TOWNS
      • Tousin Toshi for PC-9801/FM TOWNS
      • Dream Program System -SG- set2 for PC-9801/PC-88VA/FM TOWNS
      • Alice no Yakata CD for FM TOWNS
      • Dream Program System -SG- set3 for PC-9801/PC-88VA/FM TOWNS
      It cannot autodetect machine specific information. I decided PC-9801 as default. It has 4 options.
      • -v for PC-88VA (YM-2203) musics.
      • -t for FM TOWNS (YM-2612) musics.
      • -r for Rance FM TOWNS version (YM-2612) musics.
      • -s(num) for SSG relative volume (1-255%) to FM.
      If NOT -v/-t/-r options applied. It works for PC-9801 music.
    • MAKO2toVGM
      • Converter for MAKO2 format to VGM. It supports after RanceIII to Windows 3.1 era FM/SSG musics (except D.P.S -SG set3). It also converts SSG-PCM or TOWNS PCM to Windows WAVE. Autodetect file types (except RanceIV and Alice no Yakata 3). It has 5 options.
      • -n for OPN mode.
      • -x for OPM (X68000) mode.
      • -a for OPNA mode.
      • -d for first impliment (RanceIII). Correct Detune level.
      • -s(num) for SSG relative volume (1-255%) to FM.
